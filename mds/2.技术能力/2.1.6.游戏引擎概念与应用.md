## 部件系统

**是什么？在哪用？**
| 维度 | 内容 |
|------|------|
| **作用** | 组件组织功能和行为 |
| **应用场景** | 功能组织；代码复用；模块化 |
| **属性** | 位置、旋转、缩放 |
| **行为** | Start；Update；Active |
| **脚本** | 控制行为，实现逻辑 |

**会遇到哪些问题？用什么解决？**
| 问题 | 解决方向 |
|------|----------|
| **组件管理** | 优化更新；组件池；避免不必要；ECS |
| **生命周期** | 对象池；及时销毁；管理引用；避免循环引用 |

| 要点和思考方向 |
|----------------|
| 组件提供模块化；注意生命周期 |

## 序列化（Serialization）

**是什么？在哪用？**
| 维度 | 内容 |
|------|------|
| **作用** | 对象→可存储/传输格式 |
| **应用场景** | 保存；网络；资源序列化 |

| 格式 | 特点 |

|------|----------|
| **序列化性能** | 选格式；优化数据；增量；异步 |
| **版本兼容** | 版本号；数据迁移；向后兼容；测试 |

| 要点和思考方向 |
|----------------|
| 注意版本兼容；优化性能 |

## 加载

**是什么？在哪用？**
| 维度 | 内容 |
|------|------|
| **作用** | 从存储加载资源和数据 |
| **应用场景** | 资源；场景；数据加载 |

| 方式 | 说明 | 应用 |
|------|------|------|
| 同步 | 主线程一次性，界面停止 | 小资源/必须同步 |
| 异步多线程 | 不影响其他逻辑 | 大资源/可异步 |
| 伪异步 | 循环分批，精确控制 | 精确控制进度 |

**会遇到哪些问题？用什么解决？**
| 问题 | 解决方向 |
|------|----------|
| **加载阻塞** | 异步；加载进度；预加载；优化逻辑 |
| **资源管理** | 引用计数；及时释放；资源池；监控 |
| **依赖** | 依赖图；依赖注入；预加载；处理失败 |
| **性能** | 优化大小；压缩；预加载；流式 |

| 要点和思考方向 |
|----------------|
| 异步避免阻塞；资源管理；处理依赖 |
